## Описание проекта

Training Planner - это desktop-приложение для планирования и учёта тренировок с графическим интерфейсом. Программа позволяет вести журнал тренировок, фильтровать записи, а также экспортировать/импортировать данные в формате JSON.

## Основные возможности

- ✅ Добавление тренировок с валидацией данных
- ✅ Просмотр списка тренировок в таблице
- ✅ Фильтрация по типу тренировки и дате
- ✅ Сортировка по любому столбцу
- ✅ Удаление отдельных тренировок
- ✅ Экспорт и импорт данных в JSON
- ✅ Автоматическое сохранение данных
- ✅ Полная валидация ввода
- ✅ Интуитивно понятный интерфейс

## Системные требования

- Python 3.6 или выше
- Операционная система: Windows, macOS, Linux
